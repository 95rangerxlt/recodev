Steps To Reproduce:
1. Create a class like:
class InlineMethodParam
{
	String foo()
	{
		String t = bar();
		return t;
	}

	static <T> T bar()
	{
		return null;
	}
}
2. Attempt to inline t.
3. You'll get: return <String>bar();

To be legal, you need return InlineMethodParam.<String>bar();.
