Any drop-down menu from the main menu bar detaches if the window is dragged around. It disappears after the mouse button is let go but it shouldn't do this.

Reproducible: Always

Steps to Reproduce:
1. Click on a menu item such as Bookmarks and let it open.
2. While the bookmarks menu is still open, try to dragged the window around.
3.
Actual Results:  
Menu detaches and is displayed in original location until mouse button is left go at which time the menu disappears.

Expected Results:  
Menu should stay in place while window is dragged.
