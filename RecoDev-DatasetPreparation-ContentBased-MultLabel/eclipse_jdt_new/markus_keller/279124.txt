Steps To Reproduce:
1. Create a class like:

class PND {
    void foo() {
        int a = 7;

        if (a != 4 &&
            // check a is 5
            (a != 5)) {
            System.out.println(a);
        }
    }
}

2. Place the cursor before the the open bracket: (a != 5)
3. Quick-fix -> remove extra parenthesis
4. Recieve:

        if (a != 4 &&
            a != 5) {
            System.out.println(a);
        }

..with the comment destroyed.

There are other issues about comments being removed when, say, a local variable is inlined, but I believe this to be a separate case.
