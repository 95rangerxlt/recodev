The problem is NOT the final directory but the Zip-Tools TEMP directory.
Just set it to a "shorter" directory (like C:\Temp - you can find it for Winrar under Options\settings\Folders\) and not the long user directory (C:\Documents and Settings\ashenk\Local Settings\Temp) Then everything will work fine also with winrar, winzip etc.
