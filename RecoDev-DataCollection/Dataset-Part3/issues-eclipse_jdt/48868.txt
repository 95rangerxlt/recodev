20011216
Added option to avoid reporting a warning when overriding a deprecated method.
By default, such warnings are no longer reported. 
 * COMPILER / Reporting Deprecation When Overriding Deprecated Method
 *    When enabled, the compiler will signal the declaration of a method
overriding a deprecated one.
 *    The severity of the problem is controlled with option
"org.eclipse.jdt.core.compiler.problem.deprecation".
 *     - option id:       
"org.eclipse.jdt.core.compiler.problem.deprecationWhenOverridingDeprecatedMethod"
 *     - possible values:   { "enabled", "disabled" }
 *     - default:           "disabled"
