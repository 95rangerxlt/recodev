When promoting an array with predefined values to a field JDT generates illegal
java code. Example:

public class Test
{
  public void test()
  {
     String[] test = {"1", "2" };
  }
}

becomes:

public class Test
{
  private String[] test;

  public void test()
  {
     test = {"1", "2" };
  }
}

BTW: This also happens when using the quick fix "Split variable declaration".
