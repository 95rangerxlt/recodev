1. Start a new workspace
	2. Go to Preferences -> Java -> Installed JREs
	3. Add a JRE (eg. J9)
	4. Remove the default JRE (that is the one in ./eclipse/jre)
	Observe: No default JRE is selected, you can press OK and have NO JRE set.
