We will provide new functionality to easily establish project wide code conventions. The 'Clean up' action will take a set of files as input to analyze and offer to fix multiple code style issues at once. Examples of 'clean up' actions are the removal of unnecessary code or qualify all field accesses with 'this'; or assisting migration to Java 5.0 [JDT UI]
