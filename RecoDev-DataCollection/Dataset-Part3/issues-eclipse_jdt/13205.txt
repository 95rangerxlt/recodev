1. Setup JUnit
  2. In the package viewer, select class 'TestCase'.
  3. Context Menu->Copy
  4. Select compilation unit 'TestFailure.java'.
  5. Context Menu->Paste
  6. Observe that new compilation unit 'CopyOfTestCase.java' is created.

  7. Select class 'TestResult'.
  8. Context Menu->Cut, confirm both questions with 'Yes'.
  9. Select compilation unit 'TestSuite.java'.
  10. Context Menu->Paste
  11 Observe that class TestResult is created inside compilation unit
     'TestSuite.java'.
