When printing a web page with scaled-down images - although the images appear in their intended size on-screen, they appear too large on paper since they are not properly scaled down when printing.

Reproducible: Always

Steps to Reproduce:
1. Visit web page that includes images that have been scaled down using the height= and width= HTML attributes
2. Print the page
3. Observe that the images have not been correctly scaled on the printout
