The comment box to "My Polls" application in Facebook is missing. It is not missing in Firefox 3.5 or Chrome 4. No crash follows this bug; the comment box is just not there.

Reproducible: Always

Steps to Reproduce:
1. Login to Facebook
2. Make sure you have My Polls authorized (http://www.facebook.com/apps/application.php?id=61668299295&ref=appd)
3. Go into My Polls and pick any poll. The comment box is missing
Actual Results:  
The comment box is not there at all. As in completely missing.

Expected Results:  
It needs to be there.
