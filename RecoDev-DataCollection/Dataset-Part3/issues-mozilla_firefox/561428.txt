Attempt to open preference pane never opens preferences.  Keyboard input no longer works.

Reproducible: Always

Steps to Reproduce:
1. run snowleopard (latest patches)
2. Run either 64-bit or 32-bit minefield 3.7a5pre
3. Try to open preferences
Actual Results:  
Preferences never appears.  Keyboard input no longer works.  Mouse/buttons still work-- can exit from Minefield.

Expected Results:  
See preferences pane.  Keyboard input keeps working.

MacOS Snowleopard 10.6.3 w/latest security patches.

Note: I did not see this problem with Minefield 3.7a1pre - 3.7a4pre.
